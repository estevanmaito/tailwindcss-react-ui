Default button:

```js
<Button>Click</Button>
```

Disabled:

```js
<Button disabled>Disabled</Button>
```

Extended:

```js
<Button extend="shadow-lg">With shadow</Button>
```

Danger:

```js
<Button text="red-100" bg="red-600" hover="bg-red-700">
  Delete
</Button>
```

Block:

```js
<Button extend="w-full">Delete</Button>
```
